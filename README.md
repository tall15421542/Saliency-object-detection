# Saliency-object-detection

## Usage

Dependency: cmake opencv2

```
cd main  
mkdir build  
cd build  
cmake ..  
cd ../  
sh compile.sh  
./DIP [path/to/image]  
```
## option

```
cd main  
vi CMakeLists.txt
```
